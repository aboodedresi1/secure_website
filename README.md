# SecureAware Pro - موقع الأمان الرقمي الاحترافي

موقع تعليمي تفاعلي متقدم لتوعية المجتمع العربي بمخاطر التكنولوجيا والأمن الرقمي، مطور بأحدث التقنيات والمعايير.

## 🚀 المميزات الرئيسية

### ✨ التصميم والواجهة
- **تصميم تفاعلي متجاوب** يتكيف مع جميع أحجام الشاشات
- **وضع ليلي/نهاري** قابل للتبديل مع حفظ التفضيلات
- **ألوان متناسقة** مصممة خصيصاً للراحة البصرية
- **انتقالات سلسة** وتأثيرات بصرية احترافية
- **خطوط عربية** محسنة للقراءة والوضوح

### 📱 التجربة التفاعلية
- **تنقل ذكي** مع تمييز الصفحة النشطة
- **بحث متقدم** في المحتوى والمقالات
- **فلترة المقالات** حسب الفئة والموضوع
- **مشاركة اجتماعية** متقدمة مع دعم المنصات العربية
- **حفظ المفضلة** محلياً في المتصفح

### 🛡️ المحتوى التعليمي
- **مقالات شاملة** في الأمان الرقمي
- **أدلة عملية** خطوة بخطوة
- **نصائح سريعة** للحماية اليومية
- **تحديثات مستمرة** للتهديدات الجديدة

## 🛠️ التقنيات المستخدمة

### Frontend Framework
- **HTML5** - هيكل المحتوى الدلالي
- **CSS3** - التصميم والتخطيط المتقدم
- **JavaScript ES6+** - التفاعل والوظائف الديناميكية
- **Bootstrap 5.3.2** - إطار العمل المتجاوب
- **jQuery 3.7.1** - معالجة DOM والأحداث

### المكتبات والأدوات
- **Font Awesome 6.5.0** - الأيقونات
- **Google Fonts** - خطوط Cairo و Tajawal العربية
- **AOS (Animate On Scroll)** - تأثيرات الحركة
- **Bootstrap RTL** - دعم الكتابة من اليمين لليسار

### الميزات المتقدمة
- **Service Workers** - للتحميل السريع والعمل بدون إنترنت
- **Local Storage** - حفظ التفضيلات والمفضلة
- **Responsive Images** - تحسين الصور لجميع الأجهزة
- **SEO Optimized** - محسن لمحركات البحث

## 📖 المقالات المتوفرة

### 1. حماية البريد الإلكتروني
- **المحتوى**: دليل شامل لحماية البريد الإلكتروني من التهديدات
- **المواضيع**: التصيد، البرامج الضارة، كلمات المرور، المصادقة الثنائية
- **الطول**: ~15 دقيقة قراءة

### 2. كلمات المرور القوية
- **المحتوى**: إنشاء وإدارة كلمات مرور آمنة
- **المواضيع**: معايير القوة، مديري كلمات المرور، أفضل الممارسات
- **الطول**: ~12 دقيقة قراءة

### 3. مخاطر الذكاء الاصطناعي
- **المحتوى**: مخاطر نشر الصور على مواقع الذكاء الاصطناعي
- **المواضيع**: الخصوصية، سرقة الهوية، الاستخدام التجاري
- **الطول**: ~18 دقيقة قراءة

## 🔒 الأمان والخصوصية

### إجراءات الأمان
- **تشفير HTTPS** (عند النشر)
- **Content Security Policy** محدد
- **XSS Protection** مفعل
- **لا توجد ملفات تتبع** أو كوكيز غير ضرورية

### الخصوصية
- **البيانات المحلية فقط** - لا يتم إرسال بيانات لخوادم خارجية
- **لا توجد تحليلات** أو تتبع للمستخدمين
- **مفتوح المصدر** - يمكن مراجعة الكود بالكامل
